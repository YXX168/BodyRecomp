# BodyRecomp v6.3.0 更新日志

**发布日期**：2026-05-01
**更新类型**：功能增强 + 训练计划调整

## 新增功能

### 每周自动重置
- 新增每周自动重置机制，每周一开始时自动清空上一周的所有训练完成记录
- 基于 ISO 8601 周数计算，支持跨年正确识别
- 存储字段：`recomp_done_v6_week`（周数）+ `recomp_done_v6_year`（年份）
- 逻辑：App 启动时比对当前周数与存储周数，不一致则归档旧数据并清空

### 训练记录页（新增 Tab）
- 底部导航从 4 Tab 扩展为 5 Tab：训练 / 记录 / 饮食 / 超负荷 / 主题
- **月度日历热力图**：圆形格子按完成强度渐变（少→多），今天有蓝色描边高亮
- **月度统计卡片**：完成动作数 / 训练天数 / 年度累计
- **月份选择器**：左右箭头切换月份，点击标题回到当前月，未来月份不可翻
- **年度统计**：年度完成动作总数 + 训练天数
- **全年概览**：12 个月迷你月历网格（4×3 布局），每格含迷你热力点 + 当月完成数
- **图例**：少→多渐变色条说明
- 数据持久化：每月一个 SharedPreferences key（`recomp_history_YYYY_MM`），格式 `{"D_1": 5, "D_3": 8, ...}` 表示该月第1天完成了5个动作
- 每次打勾/取消动作时实时更新历史记录
- 每周重置前自动将旧数据归档到月度历史
- 当前周的完成数据也会叠加显示在月度视图中

## 训练计划变更

### 周二 — 下肢 A（股四头肌 / 前侧重点）⭐
倒蹬机 4×12-15 → 坐姿腿屈伸 3×15 → 内收机 3×15 → 坡度走40min → 卷腹 5×20

### 周四 — 下肢 B（臀腿后链 / 稳定重点）⭐
哈克深蹲 4×12 → 坐姿腿弯举 4×12-15 → 外展机 3×15 → 坡度走40min → 卷腹 5×20

## 不变计划
周一（上肢 A）、周三（上肢 B）、周五（可选循环）、周六（休息）、周日（恢复日）均无变化
