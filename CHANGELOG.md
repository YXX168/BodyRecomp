# BodyRecomp v6.4.0 更新日志

**发布日期**：2026-05-01
**基准版本**：v6.3.0（GitHub编译通过版，赛博小龙虾修复）

## 版本号修复
- pubspec.yaml 版本号更新为 6.4.0+64
- App 头部信息栏显示版本号 "v6.4.0"

## 新增功能

### 每周自动重置（v6.4.0）
- App 启动时自动检测 ISO 8601 周数
- 当进入新的一周时，先将上周完成数据归档到月度历史记录，再清空本周进度
- 存储字段：`recomp_done_v6_week`（周数）+ `recomp_done_v6_year`（年份）
- 支持跨年正确识别

### 训练记录页（v6.4.0）
- 底部导航从 4 Tab 扩展为 5 Tab：训练 / 记录 / 饮食 / 超负荷 / 主题
- **月度日历热力图**：圆形格子按完成强度渐变色（浅→深绿），今日蓝色描边高亮
- **月份选择器**：左右箭头切换月份，点击标题回到当前月，不可翻到未来月份
- **月度统计卡片**：完成动作数 / 训练天数 / 年度累计
- **年度统计**：年度完成动作总数 + 训练天数
- **全年概览**：12 个月迷你月历网格（4×3），每格有迷你热力点 + 当月完成数
- **图例**：少→多渐变色条说明
- **数据持久化**：每月一个 SharedPreferences key（recomp_history_YYYY_MM），格式 {"0": 5, "2": 8}
- 每次打勾/取消动作时实时更新历史记录
- 每周重置前自动将旧数据归档到月度历史
- 当前周的完成数据也会叠加显示在月度视图中

## 不变内容
- 训练计划数据（周一~周日）与 v6.3.0 完全一致
- 饮食营养页、渐进超负荷页、主题选择页无变化
- 主题系统、动画效果无变化
