# BodyRecomp 版本指南

## 项目概述
BodyRecomp 是一个 Flutter 健身训练追踪应用，具有动画效果和玻璃态 UI 设计。

---

## 版本历史

### v6.3.0 (当前版本) - 2026-05-01

**新增功能**：
- ✅ 训练记录页（新增 Tab）
  - 月度日历热力图（圆形格子按完成强度渐变）
  - 月度统计卡片（完成动作数/训练天数）
  - 月份选择器（左右箭头切换，点击标题回到当前月）
  - 年度统计（年度完成动作总数 + 训练天数）
  - 全年概览（12 个月迷你月历网格，4×3 布局）
  - 图例（少→多渐变色条说明）
- ✅ 每周自动重置机制
  - ISO 8601 周数计算，支持跨年
  - 重置前自动归档旧数据到月度历史
  - 存储字段：`recomp_done_v6_week` + `recomp_done_v6_year`
- ✅ 底部导航从 4 Tab 扩展为 5 Tab（训练/记录/饮食/超负荷/主题）

**训练计划调整**：
- ✅ 周二 下肢A（前侧重点）：倒蹬机 → 坐姿腿屈伸 → 内收机 → 坡度走40min → 卷腹5×20
- ✅ 周四 下肢B（后链重点）：哈克深蹲 → 坐姿腿弯举 → 外展机 → 坡度走40min → 卷腹5×20

**技术细节**：
- Flutter 3.29.0 兼容
- 新增 SharedPreferences key：`recomp_history_YYYY_MM`（月度历史）
- 版本号: 6.3.0+63

## 每周训练计划

### 周一 — 上肢 A（推力 + 后束）
胸 / 肩（前束+中束+后束）/ 肱三头

### 周二 — 下肢 A（前侧重点）
股四头 / 内收 / 有氧 / 核心

### 周三 — 上肢 B（拉力 + 手臂）
背 / 后束 / 肱二头

### 周四 — 下肢 B（后链重点）
臀腿后链 / 稳定 / 有氧 / 核心

### 周五 — 可选循环
有空就来，没空跳过

### 周六 — 休息日

### 周日 — 恢复日
任选其一，30-40 分钟，心率 110-135
