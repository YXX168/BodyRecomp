# BodyRecomp - 智能训练规划应用

一款简洁美观的 Flutter 健身训练规划应用，支持：

- 📅 周训练计划（上肢/下肢循环）
- ✅ 训练进度追踪
- 🎨 5 种主题配色（蓝/绿/粉/紫/橙）
- 🍽️ 营养建议
- 📈 进阶计划指导
- 💾 本地数据持久化

## 下载

从 [Releases](https://github.com/YXX168/BodyRecomp/releases) 页面下载最新 APK。

## 构建

```bash
flutter pub get
flutter build apk --release
```

## 技术栈

- Flutter 3.29.0
- Google Fonts
- SharedPreferences
